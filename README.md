# XDE130 故障信息与解决方案查询系统

这是根据上传文件《XDE130市场故障问题统计5月22.xlsx》生成的静态网页系统。

## 功能
- 故障关键词搜索：支持故障描述、处理方法、工作地点、整机编号等全文搜索
- 筛选：故障系统、保内/保外、工作地点、常见标签
- 故障卡片：显示故障描述、处理方法、车辆信息、源 Excel 行号
- 常见故障 Top 20 和故障系统统计
- 一键复制处理方案
- 导出当前筛选结果 CSV
- 手机/电脑自适应

## 本地运行
直接双击 `index.html` 打开即可。

## 部署到 Vercel / Netlify
1. 新建项目
2. 上传本文件夹内全部文件：`index.html`、`styles.css`、`app.js`、`data.js`
3. 框架选择 Other / Static Site
4. 构建命令留空，发布目录填写 `/` 或当前目录

## 数据说明
本系统共导入 7266 条故障记录。
