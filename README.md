# XDE130 故障智能诊断系统 V4.2 深度脱敏安全版

本版本用于公开链接场景，重点降低敏感信息暴露风险。

## 已隐藏/移除
- 服务单号、整机编号、车架号、客户/矿区/国家/具体地点
- 具体故障日期、交车日期、原始车辆小时数
- 保内/保外、Excel 源行号、导出 CSV 功能
- 原始长编号、电话、邮箱、链接等明显敏感信息

## 保留能力
- 输入故障现象，自动匹配历史案例
- 结合操作与保养手册知识输出检查步骤
- 输出可能原因、风险提醒、解决方案
- 按故障系统、常见标签筛选

注意：GitHub Pages 是公开静态网站。真正需要强安全隔离时，应升级为登录密码版或后台数据库权限版。
